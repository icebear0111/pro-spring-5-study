## 💻 Git Convention Rule

- **Commit** : `add : 챕터명 doc upload`
    - **Commit Message Title** : `add`, `fix`, `delete`
    - ex) `add : chapter01 doc upload`
- **Pull Request** : `add : 챕터명 doc upload`
    - Commit Convention과 동일 + PR template에 맞게 작성
